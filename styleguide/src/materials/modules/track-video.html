This class need to be in an iframe. ### Alternative initialization If you're a controlling type and you gotta have it your way... Alternatively you could initiate the plugin with unique or custom identifier, like so: ```javascript $(".your-iframe-selector").trackVideo(); ``` ### YouTube Quirks YT videos needs `?enablejsapi=1` within the video url for api to work. ### Vimeo and Youtube Examples (View console log to see tracking titles): ---
