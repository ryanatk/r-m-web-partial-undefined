--- notes: | All SVG's as background images must contain the following classes: `.bg-icon`, `.bg-icon--{{icon-size}}`, and `.icon--{{icon-name}}`. Sizes include `sm`, `md`, and `lg`. Note: `.icon--{{icon-size}}` must be included. When using a background icon modifier be sure that the SVG is added into the `backgrounds` directory within the `SVG` directory. ---
