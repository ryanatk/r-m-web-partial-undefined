--- notes: | There comes a time when we may not want to have a form give instant feedback when a field is invalid. For such situations, provide the following data attribute: - `data-validate-only-on-submit`: set to true or nothing, it's HTML so it doesn't matter the value Behind the scenes, when this attribute has been set on the form, it will notify all fields in the form that validation shouldn't only happen upon submit. This is done by overriding the built in invalid styling on an element. Since this is built in browser behavior, we cannot override it, but we have the ability to mask or suppress the error with the intention of showing the error when the form has been properly submitted. _Note: This only currently works on elements within a form. If there is a need to have this work on a container that isn't a form, that will have to be implemented at a later point._ ---
